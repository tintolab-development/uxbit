## 🎯 MVP V2 개요

### MVP V2 목표

반응형 웹 이커머스 사이트로, 모든 디바이스에서 최적화된 쇼핑 경험을 제공합니다. 모바일 앱이 아닌 웹 브라우저에서 완전한 기능을 제공하는 것이 목표입니다.

### MVP V2 핵심 기능

1. **홈 화면 (HOME_001)** - 단일 화면 구현
   - 헤더 영역 (로고, 검색바, 장바구니)
   - 메인 배너 캐러셀
   - 카테고리 섹션
   - 인기 상품 그리드
   - 특가 상품 섹션
   - 신상품 섹션
   - 푸터 영역

### MVP V2 vs V1 차이점

| 항목       | V1               | V2                 |
| ---------- | ---------------- | ------------------ |
| 플랫폼     | 모바일 앱 스타일 | 반응형 웹          |
| 화면 수    | 4개 화면         | 1개 화면 (홈)      |
| 네비게이션 | 하단 고정        | 헤더/푸터          |
| 레이아웃   | 세로 스크롤 중심 | 그리드 기반 반응형 |
| 이미지     | Placeholder      | 실제 이미지 URL    |

## 🎨 디자인 시스템 활용

### 사용 가능한 Custom Elements (18개)

#### 레이아웃 컴포넌트

- **tinto-app-route**: 페이지 최상단 래퍼 (Safe area, Flex 레이아웃)
- **tinto-section**: 섹션 컨테이너 (Flex, Gap, Padding)
- **tinto-wrapper**: 배경 이미지, 오버레이 지원 래퍼

#### UI 컴포넌트

- **tinto-button**: 버튼 (Primary/Secondary/Tertiary, 다양한 크기)
- **tinto-card**: 상품 카드 (이미지, 제목, 가격, 배지)
- **tinto-image**: 이미지 (Lazy loading, Aspect ratio)
- **tinto-typography**: 텍스트 (H1-H6, P, Span, 반응형 폰트)
- **tinto-badge**: 배지 (할인율, 신상품 등)

#### 인터랙션 컴포넌트

- **tinto-search-bar**: 검색바 (Debounce, Autofocus)
- **tinto-carousel**: 캐러셀 (자동 재생, 네비게이션)
- **tinto-modal**: 모달 (상품 상세)
- **tinto-toast**: 토스트 메시지
- **tinto-loading**: 로딩 스피너

#### 폼 컴포넌트

- **tinto-form-input**: 입력 필드
- **tinto-filter**: 필터
- **tinto-sort**: 정렬

## 🏠 홈 화면 상세 스펙

### 1. 헤더 영역

**컴포넌트**: `tinto-section` (horizontal, justify: space-between)

**구성 요소**:

- 로고: `tinto-typography` (variant="h1")
- 검색바: `tinto-search-bar` (size="md", full width on mobile)
- 장바구니 버튼: `tinto-button` (variant="text", icon)

**반응형**:

- 모바일: 세로 배치 (로고 상단, 검색바/장바구니 하단)
- 태블릿+: 가로 배치 (로고 좌측, 검색바 중앙, 장바구니 우측)

### 2. 메인 배너 캐러셀

**컴포넌트**: `tinto-carousel` (autoplay="true", interval="5000")

**반응형**:

- 모바일: 전체 너비, 16:9 비율
- 태블릿+: 최대 너비 제한, 중앙 정렬

### 3. 카테고리 섹션

**컴포넌트**: `tinto-section` + `tinto-card` 그리드

**구성**:

- 섹션 제목: `tinto-typography` (variant="h2")
- 카테고리 카드: `tinto-card` (이미지 + 제목)
- 그리드: 모바일 2열, 태블릿 3열, 데스크톱 4-6열

### 4. 인기 상품 섹션

**컴포넌트**: `tinto-section` + `tinto-card` 그리드

**구성**:

- 섹션 제목: `tinto-typography` (variant="h2")
- 상품 카드: `tinto-card` (이미지, 제목, 가격, 할인 배지)
- 그리드: 모바일 2열, 태블릿 3열, 데스크톱 4-5열

### 5. 특가 상품 섹션

**컴포넌트**: `tinto-section` + `tinto-card` 그리드

**구성**:

- 섹션 제목: `tinto-typography` (variant="h2")
- 할인 배지: `tinto-badge` (variant="error", 할인율 표시)
- 상품 카드: `tinto-card` (원가 취소선, 할인가 강조)

### 6. 신상품 섹션

**컴포넌트**: `tinto-section` + `tinto-card` 그리드

**구성**:

- 섹션 제목: `tinto-typography` (variant="h2")
- 신상품 배지: `tinto-badge` (variant="primary", "NEW" 표시)
- 상품 카드: `tinto-card`

### 7. 푸터 영역

**컴포넌트**: `tinto-section` (background="#F5F5F5")

**구성**:

- 회사 정보: `tinto-typography`
- 링크 목록: `tinto-typography` (variant="span")
- 저작권: `tinto-typography` (variant="span", color="#999")

## 🎯 컴포넌트 활용 전략

### 최대 활용 원칙

1. **레이아웃**: `tinto-section`으로 모든 섹션 구성
2. **그리드**: CSS Grid + `tinto-section` gap 활용
3. **카드**: `tinto-card`로 상품/카테고리 표시
4. **이미지**: `tinto-image`로 모든 이미지 처리 (lazy loading)
5. **텍스트**: `tinto-typography`로 모든 텍스트 처리
6. **배지**: `tinto-badge`로 할인율, 신상품 표시
7. **인터랙션**: `tinto-modal`로 상품 상세, `tinto-toast`로 피드백

### 컴포넌트 조합 예시

```html
<!-- 섹션 헤더 -->
<tinto-section direction="row" justify="space-between" align="center" padding="24px 16px">
  <tinto-typography variant="h2" font-size="lg" font-weight="700"> 인기 상품 </tinto-typography>
  <tinto-button variant="text" size="sm">더보기</tinto-button>
</tinto-section>

<!-- 상품 그리드 -->
<tinto-section direction="row" wrap="wrap" gap="16px" padding="0 16px 24px">
  <!-- 상품 카드들 -->
</tinto-section>
```

## 🚀 Phase별 구현 계획

### Phase 1: 기본 레이아웃 구성 (우선순위: 높음)

**목표**: 반응형 웹 기본 구조 및 레이아웃

**작업 항목**:

1. ✅ `tinto-app-route`를 body 최상단 래퍼로 설정
2. ✅ 헤더 영역 구성 (로고, 검색바, 장바구니)
3. ✅ 메인 배너 캐러셀 구성
4. ✅ 푸터 영역 구성
5. ✅ 가로 스크롤 방지 (100vw 제한)

**예상 소요**: 2시간

### Phase 2: 카테고리 섹션 (우선순위: 높음)

**목표**: 카테고리 그리드 레이아웃

**작업 항목**:

1. 카테고리 데이터 추가 (10-15개)
2. `tinto-section` + `tinto-card` 그리드 구성
3. 반응형 그리드 (모바일 2열 → 데스크톱 6열)
4. 카테고리 클릭 이벤트 (상품 필터링)

**예상 소요**: 1.5시간

### Phase 3: 상품 섹션 구현 (우선순위: 높음)

**목표**: 인기/특가/신상품 섹션 구현

**작업 항목**:

1. 인기 상품 섹션 (15개 상품)
2. 특가 상품 섹션 (15개 상품, 할인 배지)
3. 신상품 섹션 (15개 상품, NEW 배지)
4. 각 섹션별 그리드 레이아웃
5. 상품 카드 클릭 → 모달 표시

**예상 소요**: 3시간

### Phase 4: 반응형 최적화 (우선순위: 중간)

**목표**: 모든 디바이스에서 최적화

**작업 항목**:

1. 모바일 최적화 (375px ~ 767px)
2. 태블릿 최적화 (768px ~ 1023px)
3. 데스크톱 최적화 (1024px+)
4. 가로 스크롤 완전 제거
5. 터치 타겟 크기 확인 (모바일)

**예상 소요**: 2시간

### Phase 5: 이미지 최적화 (우선순위: 중간)

**목표**: 실제 이미지 URL 사용 및 최적화

**작업 항목**:

1. Unsplash/Picsum 이미지 URL로 교체
2. `tinto-image` lazy loading 활용
3. 이미지 aspect ratio 최적화
4. 로딩 상태 처리

**예상 소요**: 1시간

### Phase 6: 인터랙션 개선 (우선순위: 낮음)

**목표**: 사용자 경험 향상

**작업 항목**:

1. 상품 카드 hover 효과
2. 필터/정렬 기능 (선택적)
3. 무한 스크롤 (선택적)
4. 검색 기능 개선

**예상 소요**: 2시간

## 📈 성공 지표

### 기술적 지표

- ✅ 가로 스크롤 없음 (100vw 이하)

- 상품 카드 클릭률: 20% 이상

- 페이지 로딩 시간: 3초 이내
- 모바일 사용성 점수: 90점 이상
