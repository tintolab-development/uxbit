## **1. 핵심 개선 사항**

### **1.1 반응형 UI 구조**

| 항목               | 정의                                              | UX 기준                                |
| ------------------ | ------------------------------------------------- | -------------------------------------- |
| **브레이크포인트** | 375px (iPhone SE), 414px (iPhone 8), 768px (iPad) | 2~12칼럼 그리드, 텍스트 줄임표 처리    |
| **이미지 최적화**  | `srcset` 지원 (1x, 2x, 3x), WebP 포맷             | 100KB 이하 이미지 로딩                 |
| **터치 인터랙션**  | 터치 영역 최소 44x44px                            | 스와이프 이벤트 (슬라이드 3개 이상 시) |

### **1.2 UX 성과 지표**

| 지표                | 정의                  | 측정 방법                    |
| ------------------- | --------------------- | ---------------------------- |
| **검색바 가시성**   | 화면 상단 15% 내 위치 | Heatmap 분석 (5초 이내 인식) |
| **프로모션 클릭률** | 15% 이상              | GA4 이벤트 (`banner_click`)  |
| **A/B 테스트**      | UI 변경 효과 검증     | GA4 이벤트 기반 실험 설계    |

### **1.3 오류 UI 정책**

| 오류 코드         | 대응 정책                                | UX 구현                  |
| ----------------- | ---------------------------------------- | ------------------------ |
| **404**           | 기본 섹션 대체 + 홈 이동 버튼            | GA4 이벤트 기록          |
| **500**           | "서버 오류" UI + 3회 재시도 후 캐시 표시 | 10초 대기 후 자동 재시도 |
| **네트워크 오류** | 캐시 콘텐츠 우선 표시                    | 캐시 정책 우선           |

## **2. 화면 단위 디자인 및 UX**

### **2.1 MVP 핵심 화면**

#### **Phase 1: 핵심 UX 구현**

| 화면ID     | 주요 기능                 | UX 설계 |
| ---------- | ------------------------- | ------- |
| `HOME_001` | 프로모션 슬라이드, 검색바 |

- 슬라이드 자동 재생 (5초 간격)
- 검색바 아이콘 클릭 시 실시간 검색 활성화
- 프로모션 클릭 시 GA4 이벤트 기록 |  
  | `SEARCH_002` | 실시간 검색 (debounce 300ms) |
- 3글자 이상 입력 시 결과 표시
- 검색 결과 스크롤 시 무한 로딩 표시 |  
  | `ERROR_404_004` | 정적 오류 UI + 홈 이동 |
- "404 Not Found" 대형 텍스트 + 홈 버튼
- GA4 이벤트 기록 |

#### **Phase 2: 개인화 UX 확장**

| 화면ID             | 기능                    | 예정 시기 |
| ------------------ | ----------------------- | --------- |
| `PROMO_BANNER_003` | 개인화 추천 알고리즘    | 2024 Q2   |
| `CACHE_005`        | 스크롤 로드 범위 재조정 | 2024 Q3   |

## **3. UX 검증 결과**

- **화면 일관성**: 5개 화면ID 정상 연결
- **오류 UI**: GA4에 `error_404`, `error_500` 이벤트 정상 기록
- **터치 타겟**: 모든 클릭 영역 44x44px 이상
- **캐시 정책**: 스크롤 300px 범위 내 섹션 로드 정상
