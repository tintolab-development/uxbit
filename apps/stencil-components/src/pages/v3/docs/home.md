# 화면 와이어프레임 (Text Wireframe)

화면ID: HOME_001  
화면명: 사용자 홈 화면 (일반 사용자)

## 1) 화면 목적

- 사용자 인증 완료 후 첫 진입 화면
- 검색/프로모션/내비게이션 핵심 기능 제공
- UX/KPI 측정 및 오류 처리 정책 반영

---

## 2) 레이아웃 구조 (Top → Bottom)

- **Header**:
  - 높이: 화면 상단 15%
  - 구성: 검색바 (전체 너비), GA4 이벤트 `home_view` 기록
- **Body**:
  - 프로모션 배너 (가로 스크롤 가능)
  - 배너 개수 ≥3 시 스크롤 활성화
  - 이미지 포맷: WebP (최적화)
- **Footer**:
  - 4개 아이콘 내비게이션 (홈/검색/프로필/설정)
  - 아이콘 간 간격: 24px

---

## 3) 구성 요소 상세

### [1] 검색바

- **조건**: 항상 표시
- **사용자 액션**: 클릭 시 `SEARCH_002` 진입
- **시스템 응답**: `GET /api/search?query={}` 호출
- **접근성**:
  - 터치 타겟 44×44px 이상
  - 자동 완성 기능은 `SEARCH_002`에서 처리

### [2] 프로모션 배너

- **조건**: `GET /api/home` 응답 포함 시 표시
- **사용자 액션**: 클릭 시 `PROMO_BANNER_003` 진입
- **시스템 응답**:
  - GA4 이벤트 `banner_click` 기록
  - 실패 시 `GET /api/cache/banner` (최근 3개 캐시)
- **입력/검증**:
  - 배너 ID 유효성 검사
  - 잘못된 ID 시 `ERROR_404_004` 진입
- **접근성**:
  - 이미지 alt 텍스트 포함
  - 터치 타겟 44×44px 이상

### [3] 내비게이션 바

- **조건**: 항상 표시
- **사용자 액션**: 아이콘 클릭 시 해당 섹션 진입
- **시스템 응답**:
  - 홈 아이콘 클릭 시 `HOME_001` 재로딩
  - 설정 아이콘 클릭 시 내부 API 호출

---

## 4) 상태 관리

| 상태        | 조건                    | 시스템 응답                      | UX 디자인                         |
| ----------- | ----------------------- | -------------------------------- | --------------------------------- |
| **Loading** | `GET /api/home` 호출 시 | 2초 동안 로딩 스피너 표시        | 중앙 회전 아이콘                  |
| **Error**   | API 3회 실패 후         | `ERROR_500_005` 진입             | 서버 오류 메시지 + 새로 고침 버튼 |
| **Success** | API 정상 응답 시        | 프로모션 배너 및 내비게이션 표시 | 기본 레이아웃 유지                |
| **Empty**   | 프로모션 데이터 없음    | `GET /api/cache/home` 호출       | 캐시 데이터로 대체 표시           |

---

## 5) 인터랙션 흐름

1. 화면 진입 시 `GET /api/home` 호출
2. 정상 응답 시:
   - 검색바, 프로모션 배너, 내비게이션 표시
   - GA4 이벤트 `home_view` 기록
3. API 실패 시:
   - `GET /api/cache/home` 호출 (최근 3개 캐시)
   - 캐시 없을 시 `ERROR_500_005` 진입
4. 프로모션 배너 클릭 시:
   - `GET /api/banner/{id}` 호출
   - 실패 시 `ERROR_404_004` 진입

---

## 6) 데이터 & API 바인딩

- **주요 API**:
  - `GET /api/home` (진입 시 호출)
  - **캐시 정책**: 실패 시 `GET /api/cache/home` (최근 3개)
- **에러 핸들링**:
  - 3회 재시도 후 실패 시 `ERROR_500_005` 진입
  - 잘못된 배너 ID 시 `ERROR_404_004` 진입
- **성능 요구**:
  - 로딩 스피너 2초 이내 종료
  - 캐시 데이터는 300px 스크롤 범위 내 로드

---

## 7) 접근성 & 표준 준수

- **WCAG 2.1 AA**:
  - 터치 타겟 최소 44×44px
  - 이미지 alt 텍스트 포함
- **GA4 이벤트**:
  - `home_view`, `banner_click`, `error_404`, `error_500` 기록
