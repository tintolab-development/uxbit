# 코드 리뷰 체크리스트

> **목적**: 컴포넌트 코드 리뷰 시 확인해야 할 주요 항목들

---

## 🚨 Critical (필수 확인)

### 1. 무한 루프 방지

- [ ] `render()` 메서드에서 prop/state 변경하지 않음
- [ ] `componentDidUpdate()`에서 조건부 업데이트 수행
- [ ] 변경 감지 로직 포함 (길이 비교, 값 비교 등)

### 2. Shadow DOM 자식 요소 접근

- [ ] Shadow DOM 사용 시 자식 요소는 `componentDidLoad()`에서 수집
- [ ] `render()`에서는 수집된 데이터만 사용
- [ ] Slot 내부 요소 접근 방법 확인

### 3. Lifecycle 메서드 사용

- [ ] 초기화는 `componentWillLoad()`에서 수행
- [ ] DOM 접근은 `componentDidLoad()`에서 수행
- [ ] 업데이트는 `componentDidUpdate()`에서 조건부 수행

---

## ⚠️ Important (중요 확인)

### 4. State 관리

- [ ] State 변경은 Lifecycle 메서드에서만 수행
- [ ] 불필요한 State 업데이트 방지
- [ ] State 초기값 설정 확인

### 5. Props 검증

- [ ] 필수 Props는 `!` 표시 또는 기본값 설정
- [ ] Props 타입 검증
- [ ] 잘못된 Props 값에 대한 처리

### 6. 이벤트 처리

- [ ] 이벤트 핸들러에서 무한 루프 방지
- [ ] 이벤트 전파 제어 (preventDefault, stopPropagation)
- [ ] 이벤트 리스너 정리 (disconnectedCallback)
