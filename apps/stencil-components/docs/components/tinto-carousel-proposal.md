# tinto-carousel 컴포넌트 제안서

**작성일**: 2024년  
**상태**: 기획/디자인 승인 대기  
**우선순위**: 중간

---

## 🎯 제안 배경

현재 UXBIT 디자인 시스템에는 슬라이드 형태의 콘텐츠를 표시할 수 있는 컴포넌트가 없습니다.
이미지 갤러리, 제품 카드, 배너 등을 슬라이드로 보여주기 위한 Carousel 컴포넌트가 필요합니다.

---

## 💡 제안 내용

### 핵심 기능

1. **터치/스와이프 슬라이드** (필수)
   - 모바일에서 자연스러운 터치 제스처
   - 데스크탑에서 마우스 드래그 지원

2. **네비게이션 버튼** (옵션)
   - 이전/다음 버튼
   - 표시/숨김 제어 가능

3. **인디케이터** (옵션)
   - 현재 위치 표시
   - 클릭으로 이동 가능

4. **자동 재생** (옵션)
   - 배너 등에서 유용

- `reflect: true`로 HTML 속성과 동기화
- CSS 변수로 스타일 커스터마이징

## 🎨 디자인 제안 (디자이너 검토 필요)

### 기본 레이아웃

- 모바일: 한 번에 1개 슬라이드
- 데스크탑: 옵션으로 여러 개 표시 가능

### 네비게이션 버튼

- 위치: 슬라이드 좌우 양쪽 (오버레이)
- 스타일: 반투명 원형 버튼 + 화살표 아이콘
- 표시: 호버 시 또는 항상 (옵션)

### 인디케이터

- 타입: 점 (dot) 형태
- 위치: 슬라이드 아래 중앙
- 스타일: 작은 점들, 활성화된 점은 더 크고 진하게
