```bash
  "mcpServers": {
    "uxbit": {
      "command": "node",
      "args": ["/Users/tintolab/Desktop/uxbit/apps/mcp-server/dist/index.js"]
    }
  }

# 3. Cursor 재시작 후 테스트
```

**테스트 항목:**

- [ ] 컴포넌트 목록 조회
- [ ] 컴포넌트 검색
- [ ] 컴포넌트 상세 정보 조회
- [ ] Props 검색
- [ ] 컴포넌트 개요 문서 조회

**Phase 3 기능:**

- 파일 감시 (자동 리로드)
- 추가 검색 기능

- 통계 정보 제공

## Phase 3 상세 계획

### 3.1 파일 감시 (File Watching)

**목적:** `custom-elements.json`이나 문서 파일이 변경되면 자동으로 캐시 갱신

**구현 내용:**

- `fs.watch` 또는 `chokidar` 사용
- 파일 변경 감지 시 캐시 무효화
- 개발 중 자동 업데이트

**예상 시간:** 30분

### 3.2 추가 검색 기능

**목적:** 더 강력한 검색 기능 제공

**구현 내용:**

- 이벤트 이름으로 검색
- 메서드 이름으로 검색
- 태그명 자동완성
- 유사도 기반 검색

**예상 시간:** 1-2시간

### 3.3 성능 모니터링

**목적:** 서버 성능 추적 및 최적화

**구현 내용:**

- 요청 처리 시간 측정
- 캐시 히트율 추적
- 메모리 사용량 모니터링
- 통계 정보 제공 도구

**예상 시간:** 1-2시간

### 3.4 통계 정보 제공

**목적:** 컴포넌트 라이브러리 통계 정보 제공

**구현 내용:**

- 전체 컴포넌트 수
- Props/Events/Methods 통계
- 가장 많이 사용되는 Props
- 컴포넌트별 복잡도 지표

**예상 시간:** 1시간
