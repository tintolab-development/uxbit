## 품질 평가 섹션 추가 방법

### 1. MCP 서버로 평가 실행

Cursor에서:

```
"tinto-button 컴포넌트의 품질을 평가해줘"
```

### 2. 평가 결과를 문서에 추가

평가 결과를 위 템플릿 형식에 맞춰 문서에 추가합니다.

### 3. GitHub Issue 자동 생성

문서가 커밋되면 워크플로우가 자동으로:

- 문서를 읽어서 Issue 생성/업데이트
- 품질 평가 섹션을 자동으로 포함
- 라벨 및 담당자 자동 할당
